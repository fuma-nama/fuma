## UI Design

UI Design is important for every web developer, it's simple yet complicated enough that cannot be explained easily.

In this article, I want to cover some common mistakes made by beginners.

### Example

export const inputVariant =
  "text-neutral-50 bg-neutral-900 rounded-lg w-full p-2 text-xs placeholder:text-neutral-600 focus:outline-none focus-visible:ring-2 focus-visible:ring-blue-500";

<div className="not-prose p-4 rounded-xl bg-neutral-800 border border-neutral-600">
  <h4 className="font-medium text-xl text-neutral-50 mb-4">Create Account</h4>
  <p>Create an account on Fuma Chat, no fees required!</p>

<fieldset className="mt-4">
  <label htmlFor="name" className="text-xs text-neutral-50">
    Name
  </label>
  <input id="name" className={inputVariant} placeholder="Name" />
</fieldset>
<fieldset className="mt-4">
  <label htmlFor="password" className="text-xs text-neutral-50">
    Password
  </label>
  <input
    type="password"
    id="password"
    className={inputVariant}
    placeholder="Password"
  />
</fieldset>

<div className="flex flex-row gap-2 mt-4">
  <button className="bg-blue-500 text-neutral-50 rounded-lg px-4 py-1">Submit</button>
  <button className="bg-blue-500 text-neutral-50 rounded-lg px-4 py-1">Cancel</button>
</div>
</div>

Can you find any problems? Yes, we can find:

1. **Accessibility:** Low contrast with input placeholders.
2. **Messy:** There are more than 1 primary buttons.
3. **Spacing:** The description is too far away from title. Input paddings need some optical corrections.
4. **Redundant:** The description feels redundant, _"Create an account on Fuma Chat"_ is obviously unnecessary as the user definitely knows it. Also same for the input placeholders.

#### Optical Correction

Look at the boxes carefully.

<Image src={UIBefore} alt="boxes" className="rounded-xl" />

The padding of our first box might looks a bit off, but why?
Human's brain works different from computers. While the xy paddings are actually equal, due to font height (or bounding box), it looks different to us.

<Image src={UICompare} alt="compare" className="rounded-xl" />

In fact, it can be caused by many other factors, such as visual weight and even colors.
[This Article about Optical Adjustment](https://marvelapp.com/blog/optical-adjustment-logic-vs-designers) has explained it well.

### Correction

Now it looks much better. However, it looks dull.
